# Energy Grid Tycoon

城市能源模拟经营游戏。玩家建设发电与储能设施，调整电价，在需求高峰和随机事件中维持城市发展。

## 当前可玩内容

- 三座按顺序解锁的城市关卡
- 光伏、风电、燃气调峰与两级储能建筑
- 昼夜负荷、发电波动、电网损耗、经济和人口模拟
- 储能自动充放电、效率损耗和容量显示
- 动态城市事件
- 本地存档、自动存档、读档、重试和下一关流程
- 胜利、失败与战役选择界面
- 响应式桌面和移动端 UI

## 开发

```bash
npm install
npm run dev
npm run build
```

## 架构

- `core`：应用、会话、状态和版本化存档
- `systems`：电力、储能、经济、事件、目标等公共算法
- `gameplay`：特殊玩法扩展
- `buildings`：建筑模型、工厂、集合和运行时快照
- `data`：关卡、建筑和事件参数
- `resources`：全局素材标识
- `ui`：界面渲染与输入

完整边界与扩展规则见 `docs/ARCHITECTURE.md`。

核心原则：

- 关卡 = 参数
- 素材 = 全局资源
- 算法 = 公共模块
- 特殊玩法 = 独立扩展
- 存档 = 状态与运行时快照，不复制配置
